# This is release's readme.
# Misakiii's Crosshair

> 基于cpp和SDL3的屏幕准星显示工具，用于在屏幕显示自定义准星。

## 🛠️ 用法

### 🎯 添加新准星

> 将`png` /  `jpg` 格式图片文件放在 `./res/images/` 文件夹下。

> 在控制台使用 `open xxx.png` / `open xxx.jpg` (xxx为图片名)，打开对应图片文件并显示。

### ⌨ 对准星进行控制

> 在程序的控制台中输入相应命令以完成对准星的操作

| 命令 | 参数 | 功能 |
| --- | --- | --- |
| `help` | | 显示所有可用命令|
| `open` | `图片名` | 打开指定图片文件，图片名需包含后缀满足png或jpg格式。例如：`crosshair.png`。
| `cross` | `true`或`false` | 开关屏幕中心辅助准星 |
| `set anchor` | | 设置图片锚点到鼠标位置，后续部分操作锚点将相对于屏幕位置固定。应与`cross`设置的屏幕辅助准星对齐。|
| `move` | | 使用键盘`W` `A` `S` `D` 进行锚点的精细移动，`空格` 或窗口失去焦点停止移动。 |
| `move center` | | 将图片移至锚点与屏幕中心点对齐位置。 |
| `move mouse` | | 将图片移动至锚点与鼠标位置对齐处。 |
| `set scale` | `比例（小数）` | 设置图片的缩放比例，范围 `[0.01, 100]` 。如: `set scale 0.5` 即将图片缩小至 `0.5` 倍。 |
| `set alpha` | `不透明度（小数）` | 设置图片的不透明度， `1` 为完全不透明， `0` 为完全透明。范围 `[0, 1]` 。如 `ser alpha 0.8` ，将图片的不透明度设置为 `0.8` 。 |
| `set angle` | `旋转角度（小数）` | 将图片顺时针旋转指定角度。如： `set angle 90` ，将图片顺时针旋转 `90°` 。 |
| `set flip` | `n` 或 `h` 或 `v` 或 `hv` | 将图片翻转。 `n` 为不进行翻转， `h` 为水平翻转， `v` 为竖直翻转， `hv` 为水平竖直同时翻转。 |
| `save` | | 保存设置。 |
| `exit` | | 保存设置并退出。 |
| `help` | | 显示所有命令。 |

## 📃 json文件格式

### 1. `./res/config.json` 文件

> 用来存放上一次关闭时打开的图片，屏幕中心辅助准星是否开启。

> 当程序正常运行时会在关闭时自动保存。

#### *！！！复制时去掉注释！！！*

```json
{
    "name": "default.png",  // 上一次关闭时打开的文件
    "showCross": false      // 是否开启屏幕中心辅助准星
}
```

#### *！！！复制时去掉注释！！！*

### 2. `./res/images/xxx.json` 文件

> 存放单个图片保存的设置。

> 当程序正常运行时会在关闭时自动保存。

```json
{
    "alpha": 127,                   // 透明度，范围char [0, 255]
    "anchorPoint": {                // 锚点坐标
        "x": 342.0,
        "y": 220.0
    },
    "anchorPointK": {               // 锚点相对位置与图片大小的比例
        "x": 0.949999988079071,
        "y": 0.5412054061889648
    },
    "angle": 0.0,                   // 顺时针旋转角度
    "flip": 0,                      // 翻转。 0：无翻转 1：水平翻转 2：竖直翻转 3：水平竖直同时翻转
    "height": 813.0,                // 图片高度
    "rect": {                       // 位置和大小矩阵
        "h": 406.5,
        "w": 360.0,
        "x": 618.0,
        "y": 320.0
    },
    "scale": 0.5,                   // 缩放比例
    "width": 720.0                  // 图片宽度
}
```
#### *！！！复制时去掉注释！！！*



## 🔍 帮助

### 1. 添加新准星

> 将准星的图片（png或jpg类型），放在 **./res/images/** 路径下，打开程序使用 **open [对应文件名]** 打开图片。

#### 不要和已经存在的图片重名，会导致配置文件重叠。

### 2. 准星消失

> 可能是显示窗口最小化，在任务栏点击，将其点击就可以显示。

### 3. 打开控制台直接关闭 / 显示找不到图片文件

#### 可能原因

> 删除了 **./res/images/default.png** 导致。

#### 解决办法

>可以自己添加一个名为 **default.png** 的文件。
